# 🥁 Drum Machine

A React-based Drum Machine app that plays unique sounds when clicking on pads or pressing corresponding keyboard keys. Built to satisfy the freeCodeCamp Front End Libraries Certification project criteria.

---

## 🚀 Features

- 🎹 Clickable drum pads
- ⌨️ Keyboard-triggered sound playback
- 🔊 Audio playback with visual pad feedback
- 📃 Display showing the current sound name
- 🎛️ Clean, modern UI with responsive design
- 🔁 Sound clips with HTML5 `<audio>` elements

---

## 🛠️ Technologies Used

- **React.js**
- JavaScript (ES6+)
- HTML5 + CSS3
- HTML5 Audio API

---

## 🖱️ How to Use

1. Click a drum pad or press its associated key (e.g. `Q`, `W`, `E`, etc.).
2. The sound plays and the name is displayed on the screen.
3. Works on mobile and desktop browsers.

---

## ✅ User Stories (freeCodeCamp Spec)

- ✅ I can see an element with id="drum-machine"
- ✅ Inside "drum-machine", I can see 9 drum pads with unique audio clips
- ✅ Each drum pad contains a unique key trigger (e.g. `Q`, `W`, `E`)
- ✅ Pressing the key or clicking the pad triggers the correct audio clip
- ✅ Audio plays using the HTML5 `<audio>` element
- ✅ I can see a display showing which sound is playing
